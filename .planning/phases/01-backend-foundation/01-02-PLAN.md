  1. Create `supabase/migrations/001_initial_schema.sql` with the COMPLETE schema:

     a. Agreement number sequence:
        ```sql
        CREATE SEQUENCE agreement_number_seq START WITH 1;
        ```

     b. Agreement number generator function:
        ```sql
        CREATE OR REPLACE FUNCTION generate_agreement_number()
        RETURNS text AS $$
        BEGIN
          RETURN 'TJ-' || EXTRACT(YEAR FROM NOW())::text || '-' || LPAD(nextval('agreement_number_seq')::text, 4, '0');
        END;
        $$ LANGUAGE plpgsql;
        ```

     c. Agreements table:
        - `id` uuid PRIMARY KEY (gen_random_uuid)
        - `agreement_number` text NOT NULL UNIQUE DEFAULT generate_agreement_number()
        - `status` text NOT NULL DEFAULT 'draft' with CHECK constraint for: draft, sent, viewed, signed, completed, expired
        - `data` jsonb NOT NULL DEFAULT '{}'::jsonb (stores full AgreementData)
        - `token` text UNIQUE (for Phase 2 shareable links)
        - `token_expires_at` timestamptz (for Phase 2)
        - `created_by` uuid REFERENCES auth.users(id)
        - `created_at` timestamptz NOT NULL DEFAULT now()
        - `updated_at` timestamptz NOT NULL DEFAULT now()

     d. Auto-update trigger for updated_at:
        ```sql
        CREATE OR REPLACE FUNCTION update_updated_at()
        RETURNS trigger AS $$
        BEGIN
          NEW.updated_at = now();
          RETURN NEW;
        END;
        $$ LANGUAGE plpgsql;

        CREATE TRIGGER agreements_updated_at
          BEFORE UPDATE ON agreements
          FOR EACH ROW
          EXECUTE FUNCTION update_updated_at();
        ```

     e. Audit log table:
        - `id` bigserial PRIMARY KEY
        - `agreement_id` uuid REFERENCES agreements(id) ON DELETE SET NULL
        - `action` text NOT NULL with CHECK: created, updated, sent, viewed, signed, pdf_generated, downloaded, expired
        - `actor_type` text NOT NULL DEFAULT 'admin' with CHECK: admin, client, system
        - `actor_id` uuid (auth.users id for admin, null for client/system)
        - `ip_address` text
        - `user_agent` text
        - `metadata` jsonb DEFAULT '{}'::jsonb
        - `created_at` timestamptz NOT NULL DEFAULT now()

     f. IP and user-agent helper functions:
        ```sql
        CREATE OR REPLACE FUNCTION get_client_ip()
        RETURNS text AS $$
        DECLARE
          headers jsonb;
          forwarded text;
        BEGIN
          headers := current_setting('request.headers', true)::jsonb;
          forwarded := headers->>'x-forwarded-for';
          IF forwarded IS NOT NULL THEN
            RETURN split_part(forwarded, ',', 1);
          END IF;
          RETURN NULL;
        END;
        $$ LANGUAGE plpgsql STABLE;

        CREATE OR REPLACE FUNCTION get_user_agent()
        RETURNS text AS $$
        BEGIN
          RETURN current_setting('request.headers', true)::jsonb->>'user-agent';
        END;
        $$ LANGUAGE plpgsql STABLE;
        ```

     g. RLS policies -- ENABLE RLS on BOTH tables:

        **audit_log:**
        - INSERT for authenticated: WITH CHECK (true)
        - INSERT for anon: WITH CHECK (true)
        - SELECT for authenticated: USING (true)
        - NO UPDATE policy
        - NO DELETE policy
        (This makes audit_log append-only at the database level)

        **agreements:**
        - ALL for authenticated: USING (true) WITH CHECK (true)
        - SELECT for anon: USING (token IS NOT NULL AND token_expires_at > now())

     h. Indexes:
        - `idx_agreements_token` ON agreements(token) WHERE token IS NOT NULL
        - `idx_audit_log_agreement` ON audit_log(agreement_id)
        - `idx_audit_log_created` ON audit_log USING brin(created_at)
