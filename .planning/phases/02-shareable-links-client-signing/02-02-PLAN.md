  <name>Task 1: Create LinkShareModal with QR code, URL copy, and revoke/regenerate</name>
  <files>src/components/LinkShareModal.tsx</files>
  <action>
    Create `src/components/LinkShareModal.tsx` — a modal dialog for sharing agreement links.

    **Props interface:**
    ```typescript
    interface LinkShareModalProps {
      isOpen: boolean
      onClose: () => void
      agreementId: string
      agreementNumber: string
      existingToken: string | null
      existingExpiry: string | null
      onLinkGenerated: (token: string, expiresAt: string) => void
      onLinkRevoked: () => void
    }
    ```

    **Behavior:**
    - If `existingToken` is provided, show the existing link details (URL, QR, expiry date)
    - If no token, show a "Generate Link" button that calls `generateShareLink` from `../lib/agreements.ts`
    - After generating, call `onLinkGenerated` to update parent state

    **Layout (per user decisions — confirmation screen with copy/QR/SMS options):**
    - Modal overlay with centered white panel, rounded corners, shadow
    - Header: "Share Agreement {agreementNumber}" in forest-green, with X close button
    - QR code section: `QRCodeSVG` from `qrcode.react` wrapped in a white container with padding. Use `size={180}`, `level="M"`, `fgColor="#011c12"` (TJ Dark Green). The value is `${window.location.origin}/sign/${token}`
    - URL section: Read-only text input showing the full URL, with a "Copy Link" button next to it. On copy, use `navigator.clipboard.writeText()` and show "Copied!" feedback for 2 seconds
    - Expiry info: "This link expires on {formatted date}" in gray text
    - Revoke section: "Revoke & Regenerate" button (destructive styling — red outline). Calls `revokeLink` then `generateShareLink` to create a fresh token. Show a brief confirmation before revoking.
    - Close button at bottom: "Done" in forest-green

    **Branding:** Use Triple J colors — forest-green for primary actions, gold accent for the agreement number badge, white/gray backgrounds.

    **Do NOT include SMS send button** — Twilio integration is optional and depends on edge function deployment. The modal should focus on copy + QR which work immediately. SMS can be added later.

    **State management:** Use local useState for loading, copied feedback, and any confirmation dialogs. Import `generateShareLink`, `revokeLink` from `../lib/agreements.ts`.

    **Accessibility:** Modal should trap focus. Add `role="dialog"`, `aria-modal="true"`, `aria-labelledby` pointing to the header. Close on Escape key press and overlay click.
  </action>
  <verify>
    - `npx tsc --noEmit` passes
    - Component renders without errors when isOpen=true with a mock token
    - QRCodeSVG renders with the correct URL value
  </verify>
  <done>LinkShareModal displays QR code, copyable URL, expiry info, and revoke/regenerate — all wired to the agreement library functions</done>
